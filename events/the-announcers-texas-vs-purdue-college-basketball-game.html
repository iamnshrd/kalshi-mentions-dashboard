План на opening

  • Не фиксируйся на prematch baseline: live game state быстро меняет relevance strikes.
  • Следи за announcer flow, network style и early narrative framing.

После opening / Q&A

  • После ключевых игровых моментов repricing должен идти по clusters: replay, penalty, measurement, blowout yapping.
  • Если game script ломает prematch assumption, пересобирай view быстро и без якорения.

Переход в Q&A

  • Q&A transition not applicable in classic sports broadcast; вместо этого думай segment transitions: live play, replay, break, postgame.

Подготовка до события

  • Проверь announcer history, sport/network quirks и game-type baseline.
  • Раздели strikes на structural, replay-driven, game-state-driven и incidental chaos strikes.

Лайв-гайд

  • Перепрайсь по game state: blowout повышает incidental yap risk, close game повышает structural relevance.
  • Replay/penalty/measurement sequences важнее общей частоты по сезону.

Риски

  • Сильная зависимость от game script.
  • Broadcast/network quirks могут смещать fair value сильнее, чем кажется.
